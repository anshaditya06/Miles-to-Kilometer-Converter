# 🚀 Miles to Kilometre Converter

A simple desktop GUI application built using **Python** and **Tkinter** that converts distance from miles to kilometres.

---

## 📌 Features

* User-friendly graphical interface
* Real-time conversion with a single click
* Accurate conversion (1 mile = 1.609 km)
* Lightweight and beginner-friendly project

---

## 🖥️ Demo

Enter a value in miles and click **Convert** to instantly see the equivalent distance in kilometres.

---

## 🛠️ Tech Stack

* **Python 3**
* **Tkinter (GUI library)**

## ▶️ How to Run

1. Clone this repository:

   ```bash
   git clone https://github.com/your-username/miles-to-km-converter.git
   ```

2. Navigate to the project folder:

   ```bash
   cd miles-to-km-converter
   ```

3. Run the application:

   ```bash
   python main.py
   ```

---

## 🧠 How It Works

* Takes user input (miles) from the entry field
* Converts using the formula:

  ```
  kilometres = miles × 1.609
  ```
* Displays the result rounded to 2 decimal places
